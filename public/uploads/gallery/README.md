# Gallery Images

This directory contains images for the gallery page.

## Directory Structure

- `wrapping-paper/` - Wrapping paper designs
- `ribbon-tags/` - Ribbon tag options
- `bows/` - Bow styles

## Adding Images

1. Place your images in the appropriate category directory
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`
3. Images will automatically appear on the gallery page
4. Image titles are automatically generated from filenames (spaces and capitalization are added)

## Example

Place an image at:
```
public/uploads/gallery/wrapping-paper/holiday-red-green.jpg
```

It will appear on the gallery page with the title "Holiday Red Green"

## Tips

- Use descriptive filenames for better auto-generated titles
- Recommended image size: 800x800px or larger for best quality
- Keep file sizes reasonable for faster page loads
